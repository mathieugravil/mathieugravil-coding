Embed a dashboard using RazorFlow jQuery plugin

In this page, a RazorFlow dashboard has been embedded into a regular HTML page using the RazorFlow jQuery Plugin. This dashboard isn't embedded as an IFrame, but inside a div. Future versions of RazorFlow will allow greater control of the embedded dashboard and allow it to interact with your main web application.

You can also change the dimensions of the Dashboard to see the resizing functionality in action.
